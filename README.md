# 🚀 Học Git & GitHub - Nền Tảng Tự Động Hóa Thực Hành

![Git Learning Platform Banner](https://git-scm.com/images/logos/downloads/Git-Logo-2Color.png)

Chào mừng bạn đến với **Git & GitHub Learning Platform** – Một nền tảng học tập tương tác toàn diện, được thiết kế chuyên biệt để giúp người mới bắt đầu (và cả những lập trình viên đã có kinh nghiệm) nắm vững các khái niệm, lệnh, và tư duy từ cơ bản đến nâng cao của Git và GitHub.

Dự án không chỉ cung cấp lý thuyết mà còn đặc biệt chú trọng vào **hệ thống giả lập môi trường Terminal & File System**, mang đến cảm giác gõ lệnh thực chiến ngay trên trình duyệt mà không lo làm hỏng máy tính.

---

## ✨ Tính Năng Nổi Bật (Features)

- **📚 Lộ trình học bài bản:** Hệ thống gồm 3 Khóa học lớn từ Git Basic, GitHub Teamwork đến Git Pro (Rebase, Cherry-Pick, Conflict, Bisect).
- **💻 Giả lập Git Terminal:** Giao diện Terminal cực xịn với command engine tùy chỉnh. Hỗ trợ các lệnh như `git init`, `git add`, `git commit`, `git status`, `git log`, `git checkout`, và `git merge`.
- **📂 File Explorer Tương Tác:** Trực quan hóa thay đổi dữ liệu khi sử dụng các lệnh Terminal. Màu sắc thay đổi theo trạng thái Git Status (Xanh dương cho Add, Xanh lá cho Staged, Đỏ cho Unstaged/Deleted).
- **✅ Nhiệm Vụ & Challenge Thử Thách:** Mỗi chương học đi kèm thách thức với các Trạng thái Mục tiêu (Expected States). Hệ thống tự động so khớp đồ thị Git (Git Graph) và lịch sử thao tác để chấm điểm.
- **🔐 Hệ Thống Người Dùng:** Xác thực (Đăng ký/Đăng nhập) liền mạch. Lưu trữ lịch sử bài học cho mỗi học sinh.
- **🎨 Thiết Kế Premium:** Giao diện tối (Dark Mode default) với cấu trúc phân tử (Component-based UI) sang trọng từ Shadcn UI, phù hợp với gu thẩm mỹ của Lập trình viên hiện đại.

## 🛠️ Công Nghệ Sử Dụng (Tech Stack)

### Frontend (Client-side)
- **Framework:** React.js (Sử dụng Vite)
- **Styling:** Tailwind CSS
- **UI Components:** Shadcn UI, Lucide Icons
- **Routing:** Wouter
- **Data Fetching:** TanStack React Query

### Backend (Server-side)
- **Core:** Node.js, Express.js
- **API Spec & Validation:** Zod
- **Build tool:** TypeScript, tsx

### Database
- **ORM:** Drizzle ORM
- **Database:** PostgreSQL (Supabase Hosting)

## 🚀 Bắt Đầu Học Ngay (How to Use)

Nền tảng này được thiết kế để học trực tiếp trên Trình duyệt Web của bạn, không yêu cầu bất kì công đoạn cài đặt phần mềm nào!

1. Truy cập vào trang chủ của nền tảng.
2. Tạo một tài khoản học viên (Chỉ mất 5 giây).
3. Bắt đầu với Khóa học **"Git Cơ Bản - Từ Zero Đến Hero"**.
4. Đọc lý thuyết và gạch lệnh tương tác trực tiếp lên cửa sổ Terminal ảo bên tay phải của khóa học.

Hệ thống sẽ tự động chấm điểm và đánh dấu bài học hoàn thành cho bạn. Chúc bạn code vui!
